# Harpreet K — Product Management Portfolio

A premium, dark-mode, minimalist single-page portfolio website built for a transitioning Product Manager with a background in Supply Chain & Logistics.

## 🚀 Features

- **Minimalist, Precision-Driven Design:** Deep near-black base with a warm amber accent color, prioritizing negative space and readability.
- **Engaging Animations:** Features an orchestrated GSAP entrance sequence, scroll-triggered reveals, and a dynamic timeline line-fill animation.
- **Smooth Scrolling:** Utilizes Lenis for an inertia-based, heavy, and controlled scrolling experience.
- **Custom Micro-interactions:** Includes a custom cursor, interactive project carousels, and hover-triggered class swaps for a premium feel.
- **Responsive & Accessible:** Fully responsive design from desktop to mobile, with built-in accessibility features including `prefers-reduced-motion` support.

## 🛠 Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Custom properties, CSS Grid, Flexbox, BEM methodology)
- **Interactivity:** Vanilla JavaScript (ES6+)
- **Animation Libraries:**
  - [GSAP](https://gsap.com/) (GreenSock Animation Platform) & ScrollTrigger
  - [Lenis](https://lenis.studiofreight.com/) (Smooth scroll)

## 💻 Local Development

To run this project locally, you don't need any complex build tools. A simple local web server will do.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kharpreet007/Harpreet-K-Portfolio.git
   cd Harpreet-K-Portfolio
   ```

2. **Start a local server:**
   Using Python (which comes pre-installed on macOS/Linux):
   ```bash
   python3 -m http.server 8080
   ```
   Or using Node.js:
   ```bash
   npx serve .
   ```

3. **View the site:**
   Open your browser and navigate to `http://localhost:8080`
